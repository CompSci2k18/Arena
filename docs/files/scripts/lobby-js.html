init

function init()

Initialises the program.  Sets the value for players, and adds event listeners to the nodes that require them.  Creates an Interval to check for updates every second

checkUpdates

function checkUpdates()

Queries the server through the Lobby by receiving JSON data through an AJAX request.  If the data received has changed since the last request, update the table.  If the host has started the game, run startGame.
