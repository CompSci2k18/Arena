Arena

JavaScript client code that runs in browser.  Handles drawing of objects and, using AJAX, handles updating data by sending and receiving from the Server

Functions

init

function init()

Initialise all the global variables.

Called on ’DOMContentLoaded’ event

readyGame

function readyGame()

Checks if the game is ready to be played.

If so, starts the game.  Else, queries the server again

startGame

function startGame()

Starts the running of the game.

Calls updatePlayers, and initialises an Interval to call countdown every second

countdown

function countdown()

Displays a countdown on the canvas for countdownTimer seconds before starting the game.

To start the game, an Interval is set to call update every 16 milliseconds, and the Interval calling countdown is removed

Parameters

jqXHR reqjqXHR object containing the response information

update

function update()

The main game loop.

First checks if the game is over using isGameOver.

If not, it updates the scoreboard with updateDisplays, and draws all the objects on screen with draw

draw

function draw()

Clears the canvas, the sends calls to each Obstacle and Player object in obstacles and players respectively, asking them to draw themselves.

As we see in Player.draw, each Player handles the drawing of their Bullets

updateDisplays

function updateDisplays()

Updates the scoreboard with Player’s health and bullet counts
